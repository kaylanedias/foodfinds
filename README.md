## Objetivos

> **Consumir a API Spoonacular**
>
> **Incluir receitas com os ingredientes digitados pelo usuario**
>   
> **Enviar os ingredientes como parâmetros para a API para buscar receitas correspondentes**
>   
> **Exibir uma lista de resultados (imagens e títulos)**
>   
> **Ao clicar em uma receita, buscar e exibir os detalhes completos (instruções, ingredientes necessários, etc.).**
>

## Extras

> **Filtro por tempo de preparo, categoria e região**
>
> **Lista de ingredientes faltantes**
>
> **Paginação para organizar o Front-End**
